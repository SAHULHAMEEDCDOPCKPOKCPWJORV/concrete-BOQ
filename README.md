# Concrete BOQ (Estimation & Billing Pro) 🚧

A professional construction ERP dashboard for calculating concrete material quantities, water requirements, admixture dosages, and generating Bill of Quantities (BOQ) with billing summaries. Built with React, TypeScript, and modern CSS.

## 🚀 Live Application
**[View Live Application on Vercel](https://concrete-estimation-billing-pro.vercel.app)**
*(If your Vercel URL is slightly different, you can edit this link!)*

## 📋 Features
- **Concrete Estimator:** Calculate exact material requirements (Cement, Fine/Coarse Aggregates, Water) based on IS 456 & IS 10262 standards.
- **Mix Designs:** Supports Nominal Mixes (M5 to M20) and custom Design Mixes.
- **BOQ Generation:** Editable inline Bill of Quantities with real-time billing summary calculation.
- **Advanced Billing:** Configurable GST, Contingencies, Overhead, and Profit percentages.
- **Reports:** Download professional, company-standard PDF Reports and Excel Exports.
- **Secure Access:** Locked down with strict authentication for authorized personnel only.

## 🛠️ Technology Stack
- **Frontend:** React + TypeScript + Vite
- **Styling:** Custom CSS (Glassmorphism & Modern UI)
- **Charts:** Chart.js & React-Chartjs-2
- **Exports:** jsPDF, jsPDF-AutoTable, and SheetJS (XLSX)

## 💻 Local Development
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
